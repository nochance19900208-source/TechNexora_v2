# TechNeoxra Company Website

Company website for TechNeoxra — a software development company based in Nagoya, Japan.

## Tech Stack

- **Frontend:** React (Vite), React Router
- **Backend:** Node.js, Express

## How to Run

### 1. Install dependencies (first time only)

```bash
cd frontend
npm install

cd ../backend
npm install
```

### 2. Start backend (Terminal 1)

```bash
cd backend
npm run dev
```

Backend runs at http://localhost:8000

### 3. Start frontend (Terminal 2)

```bash
cd frontend
npm run dev
```

Frontend runs at http://localhost:5173

### 4. Open in browser

Visit **http://localhost:5173**

---

**Important:** You must be inside the `frontend` folder when running the frontend, and inside the `backend` folder when running the backend.
